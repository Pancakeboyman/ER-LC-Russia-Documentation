# 🗺️ Информация о Локации ER:LC

**Локация каждого игрока будет выглядеть примерно так:**

```json
{
  "LocationX": 1084.965,
  "LocationZ": 2302.28,
  "PostalCode": "218",
  "StreetName": "Park Street",
  "BuildingNumber": "2083"
}
```

Поля **LocationX** и **LocationZ** соответствуют координатам **X** и **Z** на одном из официальных изображений карты.

{% hint style="danger" %}
**ВАЖНО:** На данный момент примеры карт отсутствуют. Они будут добавлены сюда после официальной публикации от **PRC** или после того, как мы подготовим их самостоятельно.
{% endhint %}
